# Smart Home Energy Monitoring System

A smart energy monitoring system designed to track household power usage, estimate electricity consumption, and provide overload alerts using embedded systems and data visualization techniques.

## Features
- Real-time power monitoring
- Voltage and current tracking
- Energy consumption estimation
- High power usage alerts
- Dashboard visualization

## Technologies Used
- Arduino
- Embedded C
- Python
- Streamlit
- Tinkercad
- GitHub
